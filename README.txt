INTRODUCTION
------------

Brew haven Theme is Mobile-friendly Drupal 9/10 responsive theme.
This theme features are custom Banner, Social Menu Icon,
Footercopyright field, Site information Fields.


 FEATURES
--------
 * Responsive, Mobile-Friendly Theme
 * In built Font Awesome6
 * Custom Banner card.
 * Sticky header.
 * Footer copyright section.
 * Main Menu.
 * Site information.


INSTALLATION
------------

 * Install the Brew Haven theme as you would normally install a
   contributed Drupal theme. Visit https://www.drupal.org/node/1897420 for
   further information.


CONFIGURATION
-------------

Navigate to Administration > Appearance and enable the theme.

Available options in the theme settings:
 * Change banner image and text.
 * Change copyright text.
 * Change Social Media Links.
 * Change site information.
 * Show footer logo.
